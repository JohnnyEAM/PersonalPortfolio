Hardware & Storage

The lab is built on one Dell OptiPlex 7050 and one Dell OptiPlex 7040 , each with dedicated 2TB drives. One server acts as the primary host for live data and services, while the other serves as a secondary backup target for incremental and full backups. Networking is managed with a Gigabit switch and static IP addressing.

Virtualization & Software Stack

Planned software includes Proxmox VE or VMware ESXi for virtualization. Services include:

  • File Services: TrueNAS, SMB, NFS
  • Networking: pfSense / OPNsense firewall
  • Containers: Docker / Kubernetes
  • Backup Tools: rsync, Borg, Restic

Security & Learning Goals

The lab will host a dedicated Kali Linux VM for penetration testing, plus intentionally vulnerable environments like Metasploitable and OWASP Juice Shop. The goal is to explore:

  • Virtualization & containerization
  • Storage & backup strategies
  • Network segmentation & firewalling
  • Service deployment (file, web, DB)
  • Security hardening & offensive testing
