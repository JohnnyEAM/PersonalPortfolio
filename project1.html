Portfolio Website

A responsive and modern portfolio built with HTML, CSS, and JavaScript.

Design & Planning

The website began as a clean design concept, focusing on minimalism and accessibility. I wanted my portfolio to represent me—professional yet engaging. I desiged some simple wireframes in Figma to plan the layout and navigation.

Development

Using HTML, CSS, and JavaScript, I built responsive layouts, smooth animations, and reusable components to keep the site lightweight and professional.

Deployment

The site was deployed with GitHub Pages for free hosting. Future plans include upgrading to a custom domain and adding backend integration.
